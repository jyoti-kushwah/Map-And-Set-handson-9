MAP AND SET

Q1."Problem: You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string. 
            Input
        abcadeecfb
        
        Output:
        abcdef"
        

        let string="abcadeecfb";
        let s=new Set();
        for(let i=0;i< string.length;i++){
            s.add(string[i]);
        }

        let result="";
        for(let ch of s){
            result=result+ch;
        }
        console.log(result);

            for(let i=0;i< str.length;i++){
                if(map.has(str[i])){

                    map.set(str[i],value+1);
